# react-tweet-viewer
React Native Tweet Viewer

## Usage

### pit data(get access token and the other from dev.twitter.com)

```default.yaml
---
twitter.com:
  consumer_key: xxx
  consumer_secret: xxx
  access_token: xxx
  access_token_secret: xxx
```

### server run (listen 127.0.0.1:8080)

```
go run main
```
